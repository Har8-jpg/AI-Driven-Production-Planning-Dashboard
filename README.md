# AI-Driven-Production-Planning-Dashboard
This project develops an AI-driven production planning dashboard for manufacturing industries using deep learning and data analytics. The system integrates LSTM-based demand forecasting, workforce planning, capacity analysis, and scenario simulation to support production decision-making under uncertainty.

The project was developed using real industrial-inspired manufacturing data and focuses on improving production efficiency, workforce allocation, and capacity utilization.

Objectives
Develop an LSTM deep learning model to forecast weekly production demand.
Perform data preprocessing and exploratory data analysis (EDA).
Develop workforce and production planning analysis under uncertain demand conditions.
